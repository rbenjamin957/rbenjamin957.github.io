TaskHive

A Java Desktop Task Manager Application

About the Project

TaskHive is a Java desktop application designed to manage multiple categorized task lists. It uses MVC architecture and multiple design patterns such as Factory, Strategy, Composite, and Repository.

Features

  • Create and manage multiple lists
  • Add / Edit / Delete tasks
  • Due dates and urgency colors
  • Search and filtering
  • Local file storage
  • Java Swing GUI
